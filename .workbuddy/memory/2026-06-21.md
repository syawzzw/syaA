## 离线帖子页面评分系统
- **存储方案**：localStorage（file:// 下可用），key = `syaa_rating:{分区}/{日期}/{帖子名}`
- **帖子页面**：保存 HTML 时注入评分栏（`_inject_rating_widget()`）
  - 左侧固定竖排按钮 10~100 + 清除按钮
  - 点击评分 → 存 localStorage → 高亮当前评分 → toast 提示
  - 已有评分自动回显（高亮对应按钮）
- **分区目录页**：注入评分读取 JS（`_inject_rating_to_section_page()`）
  - 遍历所有 .post-card，从 localStorage 读取评分
  - 有评分则在 meta 区域前面加 "评分 XX ·"
- **主页导出按钮**：主页有"导出评分到文件"按钮
  - 收集所有 `syaa_rating:*` 的 localStorage 项
  - 导出为 `_ratings_export.json`（下载到浏览器下载目录）
- **GUI 导入按钮**：第6个Tab（高分收集区）新增"导入离线评分"按钮
  - `_import_offline_ratings()` 读取 JSON，从路径提取番号，匹配 DB designation
  - UPDATE av SET grade=? WHERE numbers_name=?
- **批量注入**：`_inject_ratings.py` 脚本已给 804 个已有帖子注入评分脚本
- **改动文件**：
  - gui_app.py: `_inject_rating_widget()`, `_inject_rating_to_section_page()`, `_import_offline_ratings()`, 保存HTML时调用注入, 分区页生成时调用注入
  - _regen_pages.py: 同步更新主页+分区页生成（含导出按钮+导出JS+评分读取JS）
  - _inject_ratings.py: 批量注入脚本（新文件）
