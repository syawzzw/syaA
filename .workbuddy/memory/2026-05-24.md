## 播放失败 Bug 修复（关键）
- **根因**：MySQL 有 `performer_grade` 列（在 grade 之后），SQLite 没有此列，导致列索引偏移
- MySQL: `rec[14]` = `local_path`, SQLite: `rec[13]` = `local_path`
- 代码中 `SELECT * FROM av` 后用硬编码索引取值，MySQL→SQLite 后索引全部错位
- **修复方案**：
  1. 添加 `AV_COLUMNS` 常量和 `IDX_*` 索引常量
  2. 所有 `SELECT * FROM av` 改为 `SELECT {AV_COLUMNS} FROM av`
  3. 所有 `rec[14]` → `rec[IDX_LOCAL_PATH]` 等，用命名常量替代数字
  4. 只检查存在性的查询改为 `SELECT 1`
- 涉及函数：`_rand_video`, `_play_good_video`, `_scan_115_worker`, `_check_designation_worker`, `_clean_db_worker`, `_only_one_worker`, `_rename_recursive`, `_delete_lowgrade_worker`, `_delete_keyword_worker`, `_delete_black_worker`
